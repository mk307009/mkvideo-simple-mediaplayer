# MKVideo - simple mediaplayer

Simple media player that can play video and music, its based on VideoView and MediaController. Supporting formats:
  - MP4,
  - 3GP,
  - WEBM,
  - M4A,
  - AAC,
  - MP3,
  - FLAC,
  - OGG,
  - WAV.
